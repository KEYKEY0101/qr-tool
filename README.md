# 二維碼生成器＋退貨記錄工具

倉庫／出貨工作用的網頁小工具：電腦當伺服器，手機用瀏覽器連線即可使用。

## 功能

### 📦 二維碼生成器
- 輸入內容產生二維碼（自動轉大寫），記錄保存在 PostgreSQL
- 歷史記錄：搜尋、重複產生自動累加次數、點擊重新顯示
- ⭐ 常用二維碼：加入常用後在「常用二維碼」分頁以卡片顯示
- 📷 貨物相片：每個二維碼可上傳相片（自動壓縮），一眼識別對應的貨物
- 搜尋同時涵蓋退貨記錄，點擊直接跳到該筆記錄

### ↩️ 退貨記錄
- 兩層結構：日期日誌（每天筆數＋數量統計）→ 點入單日記錄
- 每筆記錄：內容、三種數量（CTN／自訂單位（預設 PCS）／KG）、多張照片
- 點記錄顯示自動生成的二維碼，內容與數量可隨時修改
- 當日合計按單位分組統計

### 🌐 連線方式
- 電腦：`http://localhost:8100/`
- 手機（同網路）：`http://<電腦區網IP>:8100/`（頁面內建掃碼連線）
- 遠端（公司／4G）：程式啟動時自動用 NAT-PMP 請路由器開埠，
  以 `http://<公網IP>:8100/` 連線，外部連線需輸入 PIN 碼
- 防暴力破解：PIN 連錯 5 次暫停 5 分鐘，再錯 5 次完全封鎖遠端（只能在家中網路解鎖）

## 安裝

1. 安裝 [PostgreSQL](https://www.postgresql.org/download/)（資料庫與資料表會自動建立）
2. 安裝 Python 3.10+ 與依賴套件：
   ```
   pip install -r requirements.txt
   ```
3. 複製 `config.example.json` 為 `config.json`，填入資料庫密碼與自訂 PIN 碼
4. 啟動：
   ```
   python app.py
   ```
   Windows 可用 `啟動二維碼生成器.bat`（請先把裡面的 Python 路徑改成你自己的）
5. 手機連線需開放 Windows 防火牆 TCP 8100（可用 `開放防火牆_手機連線用.bat` 以系統管理員執行）

## 技術

FastAPI + uvicorn／PostgreSQL（psycopg2）／qrcode + Pillow／原生 HTML/CSS/JS（支援深色模式）
